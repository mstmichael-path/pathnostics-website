# Pathnostics Website

Homepage for Pathnostics — UTI diagnostics that help clinicians get treatment right the first time.

## Structure
- `index.html` — Live homepage (currently v9 with JMS copy rewrite)
- `report-explainer.html` — Standalone interactive Guidance UTI report walkthrough
- `index_v3.html` … `index_v9.html` — Version archives
- `css/` — Stylesheets (tokens, base, components, layout, report-explainer)
- `js/` — main.js + report-explainer.js
- `assets/` — Logos, images

## Tech
Vanilla HTML / CSS / JS + GSAP (CDN). No build step required.
